# IIT Jodhpur Word Embeddings Project

This project implements Word2Vec models (CBOW and Skip-gram) to learn word embeddings from textual data collected from IIT Jodhpur sources. It includes both a Gensim-based implementation with data scraping/preprocessing and a custom NumPy-based Skip-gram implementation.

## Project Structure

- `m25mac003_prob1.py`: Main script for data collection (web scraping, PDF extraction), preprocessing, training Gensim Word2Vec models, and visualization.
- `train_numpy_model.py`: Custom NumPy implementation of the Skip-gram model.
- `clean_corpus.txt`: Preprocessed text corpus used for training (generated by the main script).
- `check_dims.py`: Helper script to check array dimensions.
- `top_words.py`: Helper script to analyze top words.
- `print_embedding.py`: Helper script to inspect embeddings.
- `cbow_model.model` & `skipgram_model.model`: Trained Gensim models.

## Dependencies

Install the required packages using pip:

```bash
pip install numpy matplotlib nltk beautifulsoup4 scikit-learn gensim pdfplumber requests
```

## Usage

### 1. Data Collection & Gensim Models

Run the main script to scrape data, process PDFs, train Gensim models, and generate visualizations:

```bash
python m25mac003_prob1.py
```

This will:
- Scrape text from specified IIT Jodhpur URLs.
- Extract text from PDFs in the `pdf/` directory.
- Clean and normalize the text.
- Train CBOW and Skip-gram models.
- Save the trained models.
- Create t-SNE visualizations in the `Visualizations/` folder.

### 2. Custom NumPy Skip-Gram

To train the custom NumPy-based Skip-gram model on the cleaned corpus:

```bash
python train_numpy_model.py
```

## Features

- **Web Scraping**: Extracts text from IIT Jodhpur department and faculty pages.
- **PDF Processing**: handles PDF text extraction using `pdfplumber`.
- **Text Preprocessing**: Tokenization, stopword removal, and normalization using NLTK.
- **Model Training**: comparison between CBOW and Skip-gram architectures.
- **Visualization**: t-SNE plots to visualize the semantic clusters of word embeddings.
